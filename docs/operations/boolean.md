# Boolean Operations
**Boolean Operations** - basic operators to compare data.

## Supported
**Supported Types For Boolean Operations:** `int8 / int16 / int32 / int64`, `bool`, `str`

| Sign | Name |
|:-----|:-----|
| **>** | Bigger |
| **<** | Less |
| **==** | Equals |
| **!=** | Not Equals |
| **&&** | And |
| **\|\|** | Or |

## Examples:
```tpl-lang
5 < 10
100 < 200
50 == 50
1 != 2
1 == 1 && 5 < 10
15 > 20 || 2 == 2
```
