# If-Else
**If** _or_ **If - Else** provides conditional code execution. <br/>
First of all construction compiles condition and branching between code blocks. <br/>

## Syntax
> **if** _CONDITION_ **{** _code_ **};** <br/>
> **if** _CONDITION_ **{** _code_ **} else {** _code_ **};**

## Examples

```tpl-lang
if 5 < 10 {
  print("right!");
} else {
  print("what? ._.");
}
```
> right!

-----

```tpl-lang
If 10 > 5 {
  print("why??")
} else {
  print("okay");
}
```
> okay
